1. The main purpose of the TicTacToe class is to manage the state and rules of the game.
2. The makeMove function updates the board and switches players.
3. The checkWinner function determines if the game has a winner.
4. The board is stored in a 2D array of characters representing X,O, or empty.
5. The main.cpp file runs the game by repeatedly making moves until the game is over.
